My Dotfiles
----

Prerequisites:
- install zsh
- install antidote
- install nerdfonts

To install you need to run `install.sh` to create all links necessary.

You still need to change your shell with `chsh $(which zsh)`.

Includes config for:
- zsh & oh-my-zsh (powerline 10k theme)
- tmux
- vim
- git
